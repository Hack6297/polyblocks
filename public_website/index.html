Polyblocks Multiplayer Demo Website 0.1

Welcome to the Polyblocks Multiplayer demo site!
This is a public Render-hosted website for your Unity/Django project.


About the project:

1. A multiplayer backend built with Django and Django REST Framework, hosted on Render.
2. A Fun game that includes 3+ modes, such as playground, battle, and e.t.c.
3. A Unity client that connects to the multiplayer backend to sync player positions in real-time.

Features include:
  • Live multiplayer backend
  • Unity client integration
  • Parkour, battle, and sandbox worlds
